What is Nexty Platform?

Nexty Platform is a Fintech ecosystem which helps ecommerce startups to raise funds from the community. Nexty has several competitive advantages: Instant transfer, zero transfer fees and price stablisation. The Dual Cryptocurrency Confirmation System includes two cryptocurrencies: NTY & NTF which operate at the same time. The main cryptocurrency is intended for daily transaction usage. The secondary cryptocurrency is for the transaction confirmation system.
